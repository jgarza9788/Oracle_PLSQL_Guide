# Study Guide for Exam 1Z0-149: Oracle Database Program with PL/SQL

## Introduction

The Oracle Database Program with PL/SQL exam (1Z0-149) assesses your proficiency in developing applications using PL/SQL, Oracle's procedural extension for SQL. Achieving this certification demonstrates your ability to create, execute, and manage PL/SQL programs within the Oracle Database environment.

## Exam Details

- **Exam Code**: 1Z0-149  
- **Duration**: 90 minutes  
- **Number of Questions**: 65  
- **Question Format**: Multiple Choice  
- **Passing Score**: Not specified by Oracle  

## Exam Topics

The exam covers the following key areas:

1. **PL/SQL Fundamentals**
   - Understanding [PL/SQL architecture](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/plsql-overview.html)
   - Declaring and using [variables and constants](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/plsql-variables-and-constants.html)
   - Writing [executable statements](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/plsql-statements.html)
   - Interacting with the [Oracle Database Server](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/plsql-interacting-with-sql.html)

2. **Control Structures**
   - Implementing [conditional control (IF statements)](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/control-statements.html#GUID-AC44E8A4-B0CE-49F6-BBF0-56A19610FC7F)
   - Using [iterative control (LOOPs)](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/control-statements.html)

3. **Cursors**
   - Working with [implicit and explicit cursors](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/cursors.html)
   - Managing [cursor attributes](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/cursors.html#GUID-F7A7472B-8D35-4F4B-97FE-02FCD97A65E5)
   - Using [cursor FOR loops](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/cursors.html#GUID-AFEE3404-7C32-4AE7-9E10-99A14A15B53C)

4. **Exceptions**
   - [Handling exceptions](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/handling-exceptions.html)
   - [User-defined exceptions](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/handling-exceptions.html#GUID-A2F3573F-6BC1-44DA-9F70-DA5A4D9F07F3)

5. **Stored Procedures and Functions**
   - Creating and invoking [procedures and functions](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/subprograms.html)
   - Understanding [parameter modes (IN, OUT, IN OUT)](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/subprograms.html#GUID-C9A37AE5-CAD6-43D1-BD1C-714D7AB8A13B)
   - Managing [subprograms](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/subprograms.html)

6. **Packages**
   - Designing and using [packages](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/packages.html)
   - [Package state and initialization](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/packages.html#GUID-FE028F41-ED87-4FCD-A8C1-08BEB514CD2D)

7. **Triggers**
   - Creating and managing [triggers](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/triggers.html)
   - [Trigger restrictions and timing](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/triggers.html#GUID-B835BC07-B21F-41A5-9B9D-78F902C3A804)

8. **Composite Data Types**
   - Using [records](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/composite-data-types.html#GUID-8F5C6A3B-C3F0-4D98-8A23-EDB6F1D9BBFB) and [collections](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/composite-data-types.html#GUID-AB0D9E2E-3F96-4FD5-9D1A-06584A623DC3)
   - Manipulating [nested tables and VARRAYs](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/composite-data-types.html)

9. **Dynamic SQL**
   - Executing [dynamic SQL statements](https://docs.oracle.com/en/database/oracle/oracle-database/19/lnpls/dynamic-sql.html)
   - Using the [DBMS_SQL package](https://docs.oracle.com/en/database/oracle/oracle-database/19/arpls/DBMS_SQL.html)

10. **PL/SQL Optimization and Best Practices**
    - [Improving performance](https://docs.oracle.com/en/database/oracle/oracle-database/19/tgsql/tuning-plsql-applications.html)
    - [PL/SQL best practices](https://oracle-base.com/articles/misc/plsql-best-practices)
